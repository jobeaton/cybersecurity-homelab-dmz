# Cybersecurity Home Lab (Packet Tracer)

This Packet Tracer project simulates a secure internal and external network environment with:

- Cisco ASA firewall
- Internal and DMZ segmentation
- Dynamic RIP routing
- IDS Sensor to monitor internal traffic

## IDS Sensor
A PC labeled "IDS-Sensor" simulates traffic monitoring and represents an intrusion detection system like Snort or Wireshark.
